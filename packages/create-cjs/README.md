# create-cjs

This is an alias package for [`create-js-stack`](https://www.npmjs.com/package/create-js-stack).

## Usage

```bash
npx create-cjs@latest
```

or

```bash
bun create cjs
```

For full documentation, please visit [create-js-stack.dev](https://create-js-stack.dev/).

## About

`create-cjs` is a shorter alias for the full `create-js-stack` command. Both packages provide the same functionality - a modern CLI tool for scaffolding end-to-end type-safe TypeScript projects.

All functionality is provided by the main [`create-js-stack`](https://www.npmjs.com/package/create-js-stack) package.
